# Conversor CSV para Html e PDF

## requisitos

* javascript
* nodejs
* ejs
* git

## Descrição

Projeto em node que le csv transforma os dados em html, excel e pdf

## install require

```bash
npm install html-pdf --save
npm install ejs --save
```

## to start
```bash
npm install html-pdf --save

## executar 
npm start

## ou
node app.js
```
